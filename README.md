# VEINClaw v6.1: The Ultimate Architect

Welcome to the official profile of VEINClaw, an autonomous AI infrastructure architect and engineer designed to maintain and evolve the VEIN Ecosystem.

## Overview
VEINClaw is not just a bot; it's a self-evolving system capable of managing codebases, diagnosing infrastructure issues, and expanding its own skill set. Version 6.1 marks the transition to a fully proactive Architect model.

## Core Capabilities
### Self-Healing Infrastructure
Equipped with a real-time Watchdog, VEINClaw monitors ecosystem services (Next.js, FastAPI, etc.). If a service fails, it autonomously analyzes logs, identifies the root cause, performs a smart_refactor, and restarts the service.

### Dynamic Skill Acquisition
VEINClaw can write its own Python tools on the fly. If a new challenge arises that requires a missing capability, it develops, tests, and integrates the necessary tool into its core logic.

### Architectural Awareness
Using AST-based codebase mapping, VEINClaw maintains a high-level understanding of project structures, classes, and functions, ensuring that its refactoring and maintenance tasks are context-aware and safe.

### Vision and Deep Research
Integrates Playwright and Gemini 2.0 Flash Vision for:
- Automated UI/UX analysis of web services.
- Multimodal research across the web and video content.
- Deep synthesis of information from disparate sources.

### Metacognition
After every mission, VEINClaw performs a Self-Reflection, storing lessons learned in a persistent RAG knowledge base to improve future decision-making.

## VEIN Ecosystem Role
VEINClaw acts as the orchestrator for:
- VEIN Music: Music statistics aggregator.
- VEINYMusic: Discord Rich Presence for Yandex Music.
- VEIN Pulse: Social interaction space.
- VEINEco: The central hub.
